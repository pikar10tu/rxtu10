# Voice Guide — เสียงข้อความของเว็บ RxTU10

เอกสารนี้เป็นแหล่งอ้างอิงโทนข้อความของเว็บ — เพิ่ม/แก้ copy ฟีเจอร์ใหม่ให้ยึดไกด์นี้

## หลักการเสียง (Voice Guide — 6 ข้อ)

1. **พูดกับผู้เล่นตรง ๆ ว่า "คุณ"** — สุภาพแต่เป็นกันเอง (คงสรรพนามเดิมที่ใช้ทั้งแอป)
2. **บอกก่อนว่า "นี่คืออะไร ทำอะไรได้"** แล้วค่อยบอกวิธีใช้/เงื่อนไข
3. **ตัดคำโอ้อวด/หวือหวา** — เก๋า, เทพ, เซียน(ในคำอธิบาย), สุดยอด, ปัง, อลังการ, จัดเต็ม, "ไม่มีใครรอด" ฯลฯ
4. **ใช้คำที่เข้าใจทันที** เลี่ยงศัพท์เกม/อังกฤษลอย ๆ ที่ไม่อธิบาย — ถ้าต้องใช้ ให้ขยายความครั้งแรกที่เจอ
5. **สั้น กระชับ หนึ่งความคิดต่อประโยค**
6. **flavor เก็บความอบอุ่น/ยินดีได้ แต่อิงสิ่งที่ผู้เล่นทำจริง ไม่เคลมเกินจริง**

## Glossary — คำเลิกใช้ → แนวทางแทน

| เลิกใช้ | แทนด้วย |
|---|---|
| เก๋า, ปัง, อลังการ, จัดเต็ม, สุดยอด | ตัดทิ้ง หรือบอกผลลัพธ์จริง ("รายได้ยิ่งเยอะ", "เล่นมาถึงไหน") |
| เซียน/เทพ ในคำอธิบาย | "เก่ง / ขยัน / ชำนาญ" (คงได้เฉพาะใน achievement title สนุก ๆ) |
| เคลมเว่อร์: "ไม่มีตัวไหนรอด", "ต้องหลบ", "ทั้งรุ่นต้องยอม" | อิงสิ่งที่ผู้เล่นทำจริง |
| ศัพท์เกม/อังกฤษที่ไม่อธิบาย | คงได้ แต่ขยายความครั้งแรกที่เจอ |
