# Study-Center Nav — design spec

วันที่: 2026-07-03 · ขนาด: เล็ก (แก้ 2 ไฟล์ ไม่มี component ใหม่)

## เป้าหมาย

สลับตำแหน่งเมนูล่าง ให้ **Study อยู่กลาง** (จากเดิม Play อยู่กลาง) และทำปุ่ม Study เป็น
**วงกลมลอยพ้นขอบแถบ** ให้เด่นกว่าปุ่มอื่น — สื่อว่าแอปเน้น "เรียน" เป็นหลัก

## ขอบเขต

- **ลำดับใหม่ (ซ้าย→ขวา):** Home · Members · **Study** · Play · ฉัน
  (เดิม: Home · Members · Play · Study · ฉัน)
- ป้ายชื่อทั้งหมด**คงเดิม** (Home / Members / Study / Play + "ฉัน") — user ยืนยันไม่เปลี่ยน
- route/ลำดับหน้า/ป้ายอื่น **ไม่แตะ**
- ไม่มี component ใหม่ — RouterLink เดิม เพิ่มแค่คลาส

## ไฟล์ที่แก้

### 1. `src/App.vue` (template `nav#bottom-nav`)
- เรียงลิงก์ใหม่ตามลำดับข้างบน
- ปุ่ม Study เพิ่มคลาส `bn-center` (ยังเป็น `RouterLink to="/study"` ป้าย "Study" เดิม)

### 2. `src/style.css` (ชุดธีม sticker-pop — บล็อกใกล้ `#bottom-nav`/`.bn-item` ที่ redesign แล้ว)
เพิ่มสไตล์ `.bn-center`:
- ไอคอน 📚 อยู่ในวงกลม **~54px** พื้น `var(--primary)` (indigo) · ขอบหมึก 2px `var(--ink)` +
  เงาแข็งตามธีม (`--pop` หรือเทียบเท่า) · **ลอยพ้นขอบบนแถบ ~16px** (translateY ขึ้น)
- ไอคอนในวงกลมสีขาว (ตัดกับพื้น indigo)
- ป้าย "Study" ตัวหนังสือเดิมอยู่ใต้วงกลม ในแถบ
- **active** (อยู่หน้า Study `router-link-exact-active`): วงกลมเด้งขึ้นอีกนิด/ring หรือสีเข้มขึ้น —
  ใช้ transition เดิมของ `.bn-icon` เป็นฐาน
- จอกว้าง (≥ breakpoint ที่ nav หดเหลือ 560px กลางจอ): วงกลมอยู่กลางแถบเหมือนมือถือ ใช้กติกาเดิม

## ข้อควรระวัง

- **ไม่แตะ `--btm`** (ความสูงแถบ) — วงกลมล้นแค่ ~16px เฉพาะตรงกลาง ไม่ทำให้แถบสูงขึ้นทั้งอัน
  จึงไม่ชนคอนเทนต์ที่ UI-overlap audit แก้ไว้ (padding-bottom ของหน้าอิง `--btm`)
- วงกลมลอยขึ้น = ต้องเช็คว่าไม่บังคอนเทนต์ท้ายหน้า/FAB ช่วยเหลือ (z-index nav = 200)
- emoji 📚 ผ่าน `<Emoji char="📚" />` เหมือนเดิม (ห้าม emoji ดิบ)

## Verify

- `npm run build` ผ่าน
- dev: ดูทั้งจอแคบ (มือถือ) และจอกว้าง (≥560px) · วงกลม Study เด่น ลอยพ้นขอบ ไม่ล้นชนคอนเทนต์
- เช็ค active state ครบทั้ง 5 แท็บ (Study active = วงกลมเน้น, แท็บอื่น active = ปกติ)
- แตะสลับแท็บไปมา ป้าย/ไอคอนไม่เพี้ยน
